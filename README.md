# ENV variable
Rename .env.exemple to .env or create .env file

Add/modify your variable in .env file

```env
    APP_NAME="Le Flamant Bleu Mobile"
    API_PATH="http://exemple.fr:/api"
    VARIABLE_NAME="variable content"
```
